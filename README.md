# Shared Bookmarks

A simple web app to save and share bookmarks. Users can select a user, view their bookmarks, and add new ones. Each bookmark includes a title, URL, description, timestamp, and a like button.

## Features

- Select a user from a dropdown to view their bookmarks.
- Add a new bookmark with title, URL, and description.
- Bookmarks are displayed in **reverse chronological order**.
- Copy a bookmark URL to clipboard with a button.
- Like a bookmark and have the like count persist across sessions.
- Accessible interface (tested with Lighthouse).

## Technologies

- HTML, CSS, JavaScript (ES modules)
- Browser `localStorage` for storing bookmarks

## How to Run

1. Clone the repository.
2. Serve the project using a local HTTP server
